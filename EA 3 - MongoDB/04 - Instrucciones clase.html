EV N°3: Proyecto MongoDB

1. Objetivo de la sesión

Que cada grupo avance en su proyecto de BD en MongoDB Atlas. Pueden modelar una sola colección para comenzar, pero deben justificar si necesitan más.

Meta de hoy: Crear la BD, poblarla con documentos y realizar los ejercicios CRUD propuestos.

2. Prerrequisitos

  • Cuenta en MongoDB Atlas (free tier).
  • VS Code instalado y acceso a mongosh.
  • Archivos JSON de ejemplo (disponibles en esta página).

5. Comandos básicos

Referencia rápida de comandos CRUD.

Operación Comando (mongosh)
Insertar db.heroes.insertMany([...])
Buscar (Read) db.heroes.find().pretty()
Filtro db.heroes.find({clase: 'guerrero'})
Actualizar (Update) db.heroes.updateOne({nombre:'Dalia'}, {$set:{nivel:4}})
Eliminar (Delete) db.heroes.deleteOne({nombre:'Ciro'})

Notas para el Docente

Claves del éxito hoy:

  • Foco en el proceso: Valora más que logren conectar y ejecutar comandos que la complejidad del modelo.
  • Errores comunes:
    • Olvidar use bd_proyecto antes de insertar.
    • Confundir objetos {} con arrays [].
    • Intentar usar SQL mentalmente (JOINs) en lugar de documentos embebidos.
