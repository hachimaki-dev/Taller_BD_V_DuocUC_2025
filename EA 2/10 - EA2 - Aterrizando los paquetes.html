PASO 3: Pasos en español

1. Obtener categoría del cliente
2. Si es PREMIUM → 15% descuento
3. Si es NORMAL → 5% descuento
4. Sino → 0% descuento
5. Calcular: monto × porcentaje
6. Retornar resultado
Si un paso es confuso → divídelo más
10 min

PASO 4: Traduce a PL/SQL

FUNCTION calcular_descuento(
  p_cliente_id NUMBER,
  p_monto NUMBER
) RETURN NUMBER IS
  v_categoria VARCHAR2(20);
  v_porcentaje NUMBER := 0;
BEGIN
  -- Paso 1: Obtener categoría
  SELECT categoria INTO v_categoria
  FROM clientes WHERE id = p_cliente_id;
  
  -- Pasos 2-4: Determinar porcentaje
  IF v_categoria = 'PREMIUM' THEN
    v_porcentaje := 15;
  ELSIF v_categoria = 'NORMAL' THEN
    v_porcentaje := 5;
  END IF;
  
  -- Paso 5-6: Calcular y retornar
  RETURN p_monto * (v_porcentaje / 100);
END;
